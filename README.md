### Descrição do Projeto
O EcoHabit é um aplicativo móvel e plataforma web desenhado para residências urbanas que buscam reduzir seu impacto ambiental. O projeto foca em gamificar a economia de recursos, permitindo que usuários monitorem, reduzam e compitam amigavelmente no consumo de energia e água, gerando relatórios detalhados e recompensas reais através de parcerias locais.

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias e ferramentas:

### Frontend
- **[React.js](https://reactjs.org):** Biblioteca principal para a interface.
- **[TypeScript](https://typescriptlang.org):** Para tipagem segura e melhor manutenção.
- **[Tailwind CSS](https://tailwindcss.com):** Framework para estilização rápida e responsiva.
- **[React Router](https://teactrouter.com):** Gerenciamento de rotas.

### Backend (API)
- **[Node.js](https://nodejs.org):** Ambiente de execução JavaScript.
- **[Express](https://expressjs.com):** Framework web para Node. js.
- **[Prisma ORM](https://prisma.io):** ORM para manipulação do banco de dados.

### Banco de Dados & Infraestrutura 
- **[PostgreSQL](https://postgresql.org):** Banco de dados relacinal.
- **[Docker](https://docker.com):** Contrainerização do ambiente.
- **[AMS S3](https://amazon.com):** Armazenamento de arquivo/imagens.
